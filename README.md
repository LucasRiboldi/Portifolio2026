# Personal Portal

Meu portal pessoal construído com Next.js, TypeScript, TailwindCSS e arquitetura moderna.

## Estrutura

- `src/app` - Rotas e layouts da aplicação
- `src/components` - Componentes reutilizáveis
- `src/lib` - Utilitários e integrações (GitHub, Supabase, etc.)
- `src/styles` - Estilos globais e temas

## Como executar

```bash
npm install
npm run dev
```
